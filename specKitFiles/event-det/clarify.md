# Clarify – Event Details Page

## Feature ID
FEAT-EVENT-DET

## Goal
عرض صفحة تفاصيل فعالية تحتوي على جميع المعلومات الأساسية مع رابط تسجيل خارجي، وباللغة العربية، ومحسّنة للهواتف والإنترنت الضعيف.

## In Scope (MVP)
- عرض اسم الفعالية
- الوصف التفصيلي
- التاريخ والوقت
- الموقع (نصي فقط)
- اسم الجهة المنظمة
- رابط تسجيل خارجي (إن وجد)

## Out of Scope
- تسجيل داخلي
- خرائط تفاعلية
- تعليقات أو تفاعل المستخدمين

## Assumptions
- البيانات تأتي من API للقراءة فقط
- لا يوجد نظام تذاكر داخلي
